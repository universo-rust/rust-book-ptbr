# A Linguagem de Programação Rust

*por Steve Klabnik, Carol Nichols e Chris Krycho, com contribuições da Comunidade Rust*

Esta versão do texto pressupõe que você esteja usando Rust 1.90.0 (lançado em 18/09/2025) ou posterior com `edition = "2024"` no arquivo *Cargo.toml* de todos os projetos. Consulte a seção ["Instalação" do Capítulo 1](https://universorust.com.br/area-membro/livro-rust/1-1-instalacao) para instruções sobre como instalar ou atualizar o Rust, e veja o [Apêndice E](#) para informações sobre edições.

A versão HTML está disponível online em [https://doc.rust-lang.org/stable/book/](https://doc.rust-lang.org/stable/book/) e offline nas instalações do Rust feitas via `rustup` — basta executar `rustup doc --book` para abrir.

Várias [traduções](#) da comunidade também estão disponíveis.

Este texto está disponível em [formato brochura e ebook pela No Starch Press](https://nostarch.com/rust-programming-language-3rd-edition) (em inglês).
